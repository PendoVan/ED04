## Documentación de casos de uso
| Caso de uso: Acceso Institucional |                                                                           | 
|-----------------------------------|---------------------------------------------------------------------------|
| ID:                               | CUS - 01                                                                  |
| Descripción:                      | El usuario se registra con su correo institucional y código de estudiante |
| Actor:                            | Estudiante                                                                |
| Precondición:                     | -                                                                         |
| Postcondición                     |                                                                           |

| Flujo principal                                                             |
|-----------------------------------------------------------------------------|
| El usuario presiona el botón de Login                                       |
| Si no pertenece a la facultad, el sistema lo cataloga como usuario externo. |
| Rechazar si el correo ingresado no es institucional.                        |
| El sistema confirma el acceso.                                              |
|El sistema muestra mensaje de acceso con éxito.|
|El caso de uso termina.|
