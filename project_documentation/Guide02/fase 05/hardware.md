# Hardware  
En esta sección se detalla el hardware real utilizado durante el desarrollo del prototipo del **Sistema de Reservas de Canchas FISI**, incluyendo las características del equipo principal del desarrollador.

---

## 1. Equipo utilizado para el desarrollo

### **PC de escritorio (Equipo principal del desarrollador)**

- **Tipo:** PC de escritorio  
- **Procesador:** Intel Core i7 (10ma generación)  
- **Memoria RAM:** 16 GB  
- **Almacenamiento:** 1 TB (SSD/HDD según el equipo)  
- **Tarjeta de video:** Integrada de la placa (suficiente para desarrollo web)  
- **Conectividad:** Ethernet y Wi-Fi  
- **Sistema Operativo:** Windows 10 / Windows 11  
- **Periféricos utilizados:**  
  - Teclado y mouse estándar  
  - Monitor Full HD (1080p)

### Uso durante el desarrollo:
- Programación backend con FastAPI.  
- Gestión de base de datos con MySQL.  
- Construcción y prueba del frontend.  
- Ejecución del servidor Uvicorn.  
- Uso de Git, GitHub y GitHub Codespaces.  
- Modelado BPMN (Bizagi Modeler) y diseño gráfico (Canva/Draw.io).  
- Documentación técnica y creación de las fases del proyecto.

---

## 2. Equipos adicionales
No se utilizaron equipos adicionales ni PCs de la universidad durante el desarrollo del proyecto.

---

## 3. Conclusión
El hardware utilizado es suficiente para el desarrollo completo del proyecto, incluyendo backend, base de datos, frontend, pruebas y documentación.  
Las características del equipo aseguran un rendimiento fluido durante todo el ciclo de desarrollo del prototipo.
